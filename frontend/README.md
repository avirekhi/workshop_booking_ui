1. What design principles guided your improvements?
=> This re-design has been done with emphasis on increasing the clarity, functionality, and visual hierarchy. In addition, my objective is to have an interface that is straightforward and easy to understand.
Spacing, typography, and cards have been incorporated into this design. Hover, buttons, and navigational highlights have been used in this design to increase the level of interactivity.

In this case, it was essential to maintain the design as minimalistic as possible but yet visually appealing at the same time.
_________________________________________________________________________________________________________

2. How did you ensure responsiveness across devices?
=> Because this site will be mostly accessed by students using their mobile phones, I ensured that the design was able to accommodate the small screen size.

I utilized layouts that allow for flexibility, such as flexbox wrapping, ensuring that components such as cards adjust themselves according to the screen size of the viewer. I ensured that the menu bar and other parts of the page would be legible even on small screens.

Font sizes, spacings, and alignments of components remained consistent for ease of use and readability purposes.
_________________________________________________________________________________________________________

3. What trade-offs did you make between design and performance?
=> In the process of improving the user interface, my emphasis was on ensuring that the software remains light and fast. There was no use of bulky user interface libraries or animations; only basic styling using CSS was involved.

Performance improvement was achieved by avoiding some complicated animations and components, which ensured improved speed, despite having a better visual experience.
_________________________________________________________________________________________________________

4. What was the most challenging part of the task and how did you approach it?
=> The most difficult aspect of this task was making changes to enhance the user experience without altering the fundamental framework of the application in its current state.

I accomplished this by initially analyzing the existing design and pinpointing elements of the layout that could be enhanced for usability purposes. Afterward, I began making gradual enhancements like enhanced navigation and interactive cards.

Another difficult aspect of this process was maintaining a minimalist design despite adding more elements.
